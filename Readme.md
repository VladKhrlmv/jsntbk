# JS NoteBook

JS NoteBook is javascript online editor wich supports imports.

## Installation And Usage

JS NoteBook requires [Node.js](https://nodejs.org/) to run.

Install jsntbk package inside any dir

```sh
npm install jsntbk
```

To run use

```sh
npx jsntbk serve [options] [filename]
Options:
  -p, --port <number>                  port to run server on
```

## Features
- Document is saved automatically
- You can run old file of create new one
- Use function show() instead of console.log() to output anything
- You can add text cells and style it using md
- You can use React without importing it
- Every cell have access to each one before it
